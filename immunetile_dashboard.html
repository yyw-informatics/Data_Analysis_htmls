Escape Scoring — MHC-II binding escape risk per mutation

Epitope Group Escape Scores

  • Epitope-level scoring is the primary metric for escape risk assessment. It groups co-presented mutations within shared MHC-II binding cores, reflecting how the immune system actually sees antigenic changes.
  • Each bar represents a group of one or more mutations that co-occur in predicted epitopes. Singletons contain a single mutation.
  • Epop is population-weighted, using the same allele frequency panel as the per-mutation scores.

Gain Events (Gpop)

  • Gain events indicate new MHC-II binding acquired by variant peptides that the reference did not bind.

Mutation Escape Scores (Epop)

  • Epop = population-weighted escape score, computed by summing per-allele escape contributions weighted by allele frequency.
  • Red ≥ 0.5: high escape risk.
  • Yellow > 0: partial escape.
  • Green = 0: no escape.
  • Bars are scaled relative to the highest-scoring mutation.

Mutation × Allele Heatmap

  • Each cell shows E(m, a): fraction of reference MHC-II binding lost at that allele due to the mutation.
  • Cell color: red ≥ 0.75 (full loss), yellow ≥ 0.25 (partial), green = retained.
  • Columns ordered by allele frequency (highest left). Hover a cell for exact values.

TCR-Face Annotation

  • MHC-II presents a 9-residue binding core to T cells. Positions P2, P3, P5, P7, P8 face the TCR and directly influence T-cell recognition.
  • Escape modes: binding_loss = mutation disrupts MHC binding; tcr_face_risk = mutation at TCR-facing residue with non-conservative substitution (BLOSUM62 < 0); both = binding loss + TCR-face risk; unknown_register = core assignment unreliable.
  • Severity: high = non-conservative at P5 (central TCR contact); medium = non-conservative at P2/P3/P7/P8; low = conservative substitution at TCR face.
  • Risk alleles: number of HLA alleles where the mutation triggers tcr_face_risk or both escape mode.
